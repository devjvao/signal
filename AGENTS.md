# Project Rules for Subagents

Before starting any task, read `CLAUDE.md` in the project root and follow all rules defined there.

Key rules:
- Do NOT write to or modify `prompts.txt` — this is reserved for the main agent only.
- Do NOT add `Co-authored-by` trailers or any AI attribution to commits.
- All commits must follow the Conventional Commits format defined in `CONVENTIONAL_COMMIT_GUIDELINE.md`.
- Do NOT commit directly to `main`. Work only on the branch you were assigned.
